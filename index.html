Paarungsanalyse – Exterieur-Matching

⚠ Achtung: Inzucht und Doppelvererbung des Overo-Gens werden nicht berücksichtigt. Empfehlung!

Die vorgeschlagenen Hengste werden anhand genetischer Übereinstimmungen mit der ausgewählten Stute berechnet. Dabei fließen Merkmale wie Körperbau, Genetik und Struktur in die Bewertung ein. Der Score ergibt sich aus der durchschnittlichen genetischen Kompatibilität.

Für jedes Merkmal wird eine Note auf Basis genetischer Differenzen vergeben:

  • 0 Fehler = Note 1 (Exzellent)
  • 1 Fehler = Note 2 (Sehr gut)
  • 3 Fehler = Note 3 (Gut)
  • 4+ Fehler = Note 5 (Ausreichend)

Aus allen Merkmalen wird dann der Durchschnitt berechnet, um die beste und schlechteste Note (Best Case / Worst Case) zu bestimmen. Diese bilden den finalen Mittelwert für das mögliche Fohlen ab.
